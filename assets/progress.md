## How to Use This Tracker

- Mark **Introduced** when first exposed to a concept.
- Mark **Practiced** after repeated implementation.
- Mark **Comfortable** when solving typical problems unaided.
- Mark **Can Modify** when adapting examples to new contexts.
- Mark **Can Build** when using the concept in a complete artifact.
- Mark **Can Teach** when clearly explaining to another learner.

Keep notes brief but specific.

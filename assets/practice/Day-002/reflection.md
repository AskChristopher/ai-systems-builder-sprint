# Day 002 Reflection

## What did I learn?

- I learned truth value testing in Python and how values like `0`, `""`, `[]`, and `None` evaluate to `False`.
- I practiced reading boolean outputs and translating them into plain-English meaning.
- I reinforced how `and`, `or`, and `not` change conditional results.

## What surprised me?

- Small values and empty containers can control program flow without writing explicit `== True` checks.
- The output was simple, but understanding what each `True` or `False` meant took deliberate thinking.

## What mistakes did I make?

- I made an indentation error and had to fix block structure before the script worked correctly.
- I initially focused on printing results instead of interpreting what those results meant.

## What remains confusing?

- I still want faster intuition for combining multiple boolean conditions in one expression.

## What should I practice next?

- Learn and practice exceptions next, especially `try`/`except` and reading traceback messages.
- Write small exercises that combine conditionals with input validation and exception handling.

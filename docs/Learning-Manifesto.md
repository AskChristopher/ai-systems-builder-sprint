## What I Believe About Learning

### Learning Beliefs

- [ ] Learning is an active process, not passive consumption.
- [ ] Building real systems accelerates understanding.
- [ ] Reflection turns experience into insight.
- [ ] Teaching/documenting cements mastery.
- [ ] Consistency compounds into expertise.

### Anti-Patterns I Will Avoid

- [ ] Endless tutorial loops without shipping.
- [ ] Perfectionism that prevents iteration.
- [ ] Hiding unfinished work instead of learning publicly.
- [ ] Ignoring feedback and data from outcomes.

# ai-systems-builder-sprint

A sprint repository for building AI systems. This repo is organized for daily/weekly tracking, Python projects, notes, resources, and templates.

## Structure

- `docs/` – Documentation
- `daily/` – Daily logs and updates
- `weekly/` – Weekly summaries and reviews
- `python/` – Python scripts and utilities
- `projects/` – Project folders
- `notes/` – General notes
- `resources/` – Reference materials and links
- `templates/` – Reusable templates
